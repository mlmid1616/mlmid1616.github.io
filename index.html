GROUP WORK

Style Privé is a clean and responsive web page that serves to help users find the perfect stylist nearby with as little effort as possible. You can search, vet, and, after signing in, book a stylist.

The frontend is built using React, Javascript, Redux, Webpack, and CSS. Focal libraries include react-tabs, redux-form, react-router, and lodash.

This web application was built alongside three excellent developers from Cloud District

  • RESPONSIVE & ROLLING HOMEPAGE

    The homepage is designed so that users can easily process the information we want to convey. Using minimalist photos and a unique font, a user can understand everything Style Privé has to offer in a matter of seconds, regardless of their window size.

  • INTUITIVE AND STYLED TABS

    A user can declare the event they need a stylist for. Then, if the event a personal one, they can declare their preferred gender, the type of styling they would like, and finally location, date, and hour. This gives a list of all the stylists available for the type of service and who have marked themselves as available.

  • COMFORTABLE PERSONAL PORTAL

    After logging in, a user can access their private zone to see past stylists, edit their profile, or change their password. It was our goal to make this route extremely easy to navigate.
